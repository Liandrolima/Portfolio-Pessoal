Cursos

Programação Full Stack

Instituição: Infinity School

Este curso de Programação Full Stack oferece uma formação abrangente em desenvolvimento web, capacitando os alunos a criar e gerenciar aplicativos completos. A formação cobre tanto o desenvolvimento front-end quanto o back-end, permitindo aos alunos entender e construir interfaces de usuário interativas e robustas, além de criar e gerenciar servidores, APIs e bancos de dados. Entre os tópicos abordados estão HTML, CSS, JavaScript, frameworks modernos como React e Node.js, e bancos de dados relacionais e não relacionais. Ao final do curso, os alunos estarão aptos a desenvolver, testar e implantar aplicações web completas, integrando todos os aspectos do desenvolvimento moderno.

Bacharelado em Admnistração

Instituição: Universidade Ateneu

O bacharelado em Administração de Empresas é uma graduação voltada para a formação de profissionais capazes de gerenciar organizações de diversos setores. O curso abrange uma ampla gama de disciplinas, incluindo finanças, marketing, recursos humanos, logística, contabilidade e gestão estratégica. Durante a graduação, desenvolvem-se habilidades em liderança, tomada de decisão, análise de dados e compreensão de processos organizacionais.

Projeto: Registro de Informações Pessoais

Este projeto HTML cria uma página web simples contendo um formulário de registro para coletar informações pessoais do usuário. Projeto feito no curso de programação full stack na Infinity School.

O objetivo deste projeto é fornecer um formulário funcional e estilizado para o registro de informações pessoais, que pode ser usado em diversos contextos, como em páginas de cadastro, pesquisas ou contato. O formulário é simples e direto, assegurando a coleta das informações essenciais do usuário.
